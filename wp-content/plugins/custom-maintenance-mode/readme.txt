== Description ==

Afghan Custom Maintenance Mode is a customizable plugin that allows you to display a personalized maintenance mode page for your WordPress website. You can add a custom message, upload a background image, display a countdown timer, and include an email subscription form for visitors.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/afghan-custom-maintenance-mode` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Settings** > **Maintenance Mode** to configure your custom settings.
4. Enable the maintenance mode by toggling the option in the settings and set the countdown timer, message, and background image.

== Frequently Asked Questions ==

= How do I customize the maintenance mode message? =

You can customize the maintenance message by going to the **Settings** > **Maintenance Mode** page in the WordPress dashboard.

= Can I set a countdown timer for when the site will be live again? =

Yes, you can add a countdown timer by entering the number of hours in the settings page under **Countdown Timer**.

= Can I upload my own background image? =

Yes, the plugin allows you to upload a custom background image through the settings page.

= Does the plugin support email subscription forms? =

Yes, you can integrate an email subscription form (e.g., using MailChimp) by adding the form shortcode in the settings page.
